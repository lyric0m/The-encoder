\doxysubsection{Detailed Description}
Программа для шифрования и дешифрования текстовых файлов с использованием различных гаммовых методов. 

Программа поддерживает несколько методов шифрования, использует гаммирование с блочным шифром и может добавлять байт длины или контрольную сумму в конец зашифрованного текста. Для генерации гаммы могут использоваться различные факторы, такие как текущее время или набор символов.
